#  Security Policy

## Supported Versions
Since this is a curated list of cybersecurity resources and not executable software, versioning applies primarily to the structure and content of the repository. 

| Version | Supported          |
| ------- | ------------------ |
| v1.x.x  | :white_check_mark: |

## Reporting a Vulnerability
If you discover a security vulnerability within this repository (e.g., a malicious link, compromised tool, or dangerous command presented as safe), please do NOT open a public issue. 

Instead, please send an email directly to the maintainer or reach out via LinkedIn. All security-related reports will be addressed within 48 hours.
